manatus API
============

**manatus** is a library in addition to a CLI tool. Classes may be imported into other programs and used separately
from the CLI.

+-------------------------+----------------------------------+
| manatus.scenarios       | Parser and record classes        |
+-------------------------+----------------------------------+
| manatus.source_resource | Result data class                |
+-------------------------+----------------------------------+
| manatus.maps            | Default transformation functions |
+-------------------------+----------------------------------+
| manatus.organizations   | Data provider abstraction class  |
+-------------------------+----------------------------------+
| manatus.exceptions      | Manatus specific exceptions      |
+-------------------------+----------------------------------+
